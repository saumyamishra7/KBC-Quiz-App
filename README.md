# KBC Quiz App 

## Description
An interactive KBC-style quiz application built using HTML, CSS, and JavaScript.
The app includes multiple-choice questions, score tracking, and an engaging user interface inspired by the KBC quiz format.

## Features
- Multiple-choice quiz questions
- Interactive and user-friendly UI
- Real-time score display
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Start playing the quiz
